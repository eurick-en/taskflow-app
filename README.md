# TaskFlow App 🚀

Aplicativo mobile desenvolvido com React Native para gerenciamento de tarefas.

## 📱 Funcionalidades

- Login de usuário
- CRUD completo de tarefas
- Filtro por status (pendente / concluída)
- Persistência com AsyncStorage
- Consumo de API (frases motivacionais)
- Tema Dark/Light

## 🛠️ Tecnologias

- React Native
- TypeScript
- Context API
- AsyncStorage
- Axios

## 📸 Telas

- Login
- Home
- Lista de Tarefas
- Formulário de Tarefa
- Configurações

npm install
npx expo start
